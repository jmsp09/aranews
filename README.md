# Aranews

Aranews is an Android application designed to keep users updated with the latest news from the **Aragón Open Data** portal.

## Features

- **Real-time Updates**: Fetches the latest news using the Aragón Open Data API.
- **Offline Support**: Stores news locally using **Room/SQLite** for quick access even without an internet connection.
- **Clean UI**: Simple and intuitive interface for a seamless reading experience.
- **Privacy Focused**: No personal data is collected or stored.

## Project Structure

- `app/`: Contains the Android application source code.
- `docs/`: Contains the project documentation, including the Privacy Policy.
